Payment Provider VCSWeb

This module add VCS (Virtual Card Services), now PayGate as a payment provider. The documentation for the gateway API can be found at https://www.vcs.co.za/interfaces/Documents/VirtualOnline/ccForm.pdf VCS has been acquired by Paygate.co.za. This module supports the older VCS API.
